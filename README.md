# Smart Heritage Hebron

A modern, interactive website showcasing the transformation of the Old City of Hebron into a smart heritage destination. This project combines historical authenticity with futuristic technology to create an immersive digital experience.

## 🌟 Features

### 1. Hero Section
- Full-screen cinematic introduction with animated background
- Floating particle effects
- Smooth scroll-to-explore functionality
- Call-to-action buttons for exploration

### 2. Interactive Map
- Clickable location pins on a stylized map of Hebron's Old City
- Pop-up modals with historical information
- Past vs. present image comparison
- AR experience previews for each location

### 3. AR Experience
- Showcases augmented reality features
- Floating UI elements and holographic effects
- Interactive feature cards
- Animated scanning visuals

### 4. Smart Market
- E-commerce layout for traditional handmade products
- Product categories: Glass, Ceramics, Embroidery
- Each product includes stories and artisan information
- Video preview functionality
- Statistics dashboard showing market impact

### 5. Night Experience
- Dark mode design with starry background
- Smart lighting visualization
- Cultural event showcase
- Dynamic temperature and ambiance displays

### 6. Heritage Protection Dashboard
- Real-time building condition monitoring
- Interactive charts and graphs (using Recharts)
- IoT sensor data visualization
- Alert system for maintenance needs
- AI-powered analytics display

### 7. VR Tour
- 360° immersive experience preview
- Scene navigation with hotspots
- Multiple location views
- Interactive controls

### 8. Responsive Navigation
- Sticky navigation bar with scroll effects
- Mobile-friendly menu
- Smooth scroll-to-section functionality

## 🎨 Design Features

- **Cinematic Style**: High-end, futuristic design with smooth animations
- **Color Palette**: Warm golden amber tones with stone textures
- **Typography**: Clean, modern font hierarchy
- **Animations**: Motion/React for smooth transitions and parallax effects
- **Responsive**: Mobile-first design approach

## 🛠 Technologies Used

- **React**: Component-based UI framework
- **React Router**: Navigation and routing
- **Motion (Framer Motion)**: Advanced animations
- **Recharts**: Data visualization and charts
- **Tailwind CSS**: Utility-first styling
- **Lucide React**: Icon library
- **TypeScript**: Type-safe development

## 📱 Responsive Design

The website is fully responsive and optimized for:
- Desktop (1920px+)
- Laptop (1024px - 1920px)
- Tablet (768px - 1024px)
- Mobile (320px - 768px)

## 🚀 Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
